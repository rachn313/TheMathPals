Sequences:

<Definitions>

{an} = {a1, a2, a3, …, an, ...}

Sequence ({an}) : a list of numbers written in an explicit order

Terms ( a1, a2, a3, …, an, …): the numbers that make up the sequence

nth term = an

Finite sequence: a sequence that has a finite domain

Infinite sequence: a sequence with domains that are infinite subsets of the positive integers

<Defining sequences>

Explicitly defined: defined in terms of n

        e.g.    an = 2n/(n+3)

Recursively defined: defined by relating it to previous terms

        e.g.   a1 = 7

                an  = an-1 + 3 for all n ≥ 2

<Arithmetic and geometric sequences>

Arithmetic sequence: a sequence that can be written in the form {a, a + d, a + 2d, …, a + (n-1)d, … } for some constant d (the common difference).

* Each term can be obtained recursively from preceding term by adding d:

         an  = an-1 + d for all n ≥ 2

Geometric sequence: a sequence that can be written in the form {a, a 𝗑 r, a 𝗑 r2, … , a 𝗑 r2, … } for some nonzero constant r (the common ratio).

        * Each term can be obtained recursively from preceding term by multiplying by r:

         an  = an-1 𝗑 r for all n ≥ 2

<Limit of a sequence>

Limit (L): the sequence an  has limit L as n approaches infinity if, given any positive number 𝜖, there is a positive number M such that for all n > M we have ⎸an  - L⎹ < 𝜖.

*We say that a sequence “converges to L”

*Sequences without limits diverge

Properties of Limits

If L and M are real numbers and limn→ ∞ an = L and limn→ ∞ bn = M

  1. Sum rule: limn→ ∞ (an + bn) = L + M
  2. Difference rule: limn→ ∞ (an - bn) = L - M
  3. Product rule: limn→ ∞ (an bn) = L 𝗑 M
  4. Constant multiple rule:  limn→ ∞ (c 𝗑 an) = c 𝗑 L
  5. Quotient rule: limn→ ∞ (an / bn) = L / M,    M ≠ 0

e.g. Find the limit of the sequence an = (2n-1)/n

         limn→ ∞ an = limn→ ∞ (2 - 1/n) = limn→ ∞ (2) - limn→ ∞ (1/n) = 2 - 0 = 2

<Special theorems>

Sandwich Theorem: If  limn→ ∞ an =  limn→ ∞ cn = L, and if there is an integer N for which an ≤ an ≤ cn for all n > N, then limn→ ∞ an = L.

L’Hopital:

Limits of the following forms are called indeterminate:

0/0, ∞/∞, 0*∞, ∞-∞, 00, 1∞, ∞0 

To find the limit of an indeterminate form of the type 0/0 or ∞/∞, we apply L’Hopital’s Rule which takes the derivative of the functions in the numerator and denominator. In the following, a is a finite number. The rule has several parts:

  1. If limx->af(x)= limx->ag(x)= 0 and if limx->a f’(x)/g’(x) exists, then:

        limx->a f(x)/g(x)= limx->a f’(x)/g’(x)

If limx->a f’(x)/g’(x) does not exist, then L’Hopital’s Rule cannot be applied.

2. If limx->af(x)= limx->ag(x)= ∞, and limx->a f’(x)/g’(x) exists, then:

limx->a f(x)/g(x)= limx->a f’(x)/g’(x)

3. If limx->∞f(x)= limx->∞g(x)=0 and limx->∞f’(x)/g’(x) exists then:

 limx->∞f(x)/g(x)= limx->∞f’(x)/g’(x)

4. If limx->∞f(x)= limx->∞g(x)= ∞ and limx->∞f’(x)/g’(x) exists then:

 limx->∞f(x)/g(x)= limx->∞f’(x)/g’(x)

If applying L'Hopital's Rule to a function, we obtain 0/0 or ∞/∞ then we can repeat the process until the form obtained is no longer indeterminate.

Intermediate forms such as 0*∞, ∞-∞, 00, 1∞ or ∞0  must be transformed to 0/0 or ∞/∞ so L’Hopital’s Rule can be applied.

Improper integrals:

There are two types of improper integrals:

  1. Improper integrals in which one or both of the limits of integration is infinite
  2. Improper integrals in which f(x) has a discontinuity (becoming infinite)

For Type 1 (integrands with infinite limit/s), use infinite integration limits to find the value of improper integral.

A.  If f(x) is continuous on [a, ∞), then

        ∫∞a f(x) dx = limb→ ∞ ∫ba f(x) dx

B.  If f(x) is continuous on (-∞, b}, then

        ∫-∞b f(x) dx = lima→ - ∞ ∫ba f(x) dx

C.  If f(x) is continuous on (-∞, ∞), then

        ∫-∞∞ f(x) dx = ∫-∞c f(x) dx + ∫c∞ f(x) dx

       (where c is any real number)

In parts A & B:

  • The improper integral converges if the limit is finite (and the value of the limit is that of the improper integral)
  • The improper integral diverges if the limit fails to exist

In part C:

  • The improper integral converges if both integrals on the right-hand side converge
  • Otherwise, it diverges

Ex 1. Does ∫1∞ dx/x converge or diverge?

        ∫1∞ dx/x =  limb→ ∞ ∫b1 dx/x

                =  limb→ ∞ ln x ]b1

                =  limb→ ∞ (ln b - ln 1) = ∞

For Type 2 (integrands with infinite discontinuities), use one-sided limits to find the value of improper integral.

A. If f(x) is continuous on (a, b], then

        ∫ba f(x) dx = limc→ a+ ∫bc f(x) dx

B. If f(x) is continuous on [a, b), then

        ∫ba f(x) dx = limc→ b- ∫ca f(x) dx

C. If f(x) is continuous on [a, c) ∪ (c, b], then

        ∫ba f(x) dx =  ∫ac f(x) dx + ∫cb f(x) dx

In parts A & B:

  • The improper integral converges if the limit is finite (and the value of the limit is that of the improper integral)
  • The improper integral diverges if the limit fails to exist

In part C:

  • The improper integral converges if both integrals on the right-hand side converge
  • Otherwise, it diverges

Test for Convergence and Divergence

Comparison Test:

        Let f and g be continuous on [a, ∞) with 0 ≤ f(x) ≤ g(x) for all x ≥ a.

  1. ∫∞a f(x) dx   converges if   ∫∞a g(x) dx   converges.
  2. ∫∞a g(x) dx   diverges if   ∫∞a f(x) dx   diverges.
